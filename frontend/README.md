# Todo List - Frontend

React frontend for the Todo List application.

## Requirements

- Node.js 18+
- Backend server running on port 8081 (see `backend/`)

## Setup

```bash
npm install
```

## Development

Start the dev server (proxies `/api` to backend on port 8081):

```bash
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
```
